# Material UI playground

This playground is a great tool where you can play with Material UI components.

## Controls

Each story has controls where you can modify some props of the components.

**Controls** tab is located at the bottom/right of the screen.

## Story

If you are using **Canvas** tab, you can copy source code from story tab located at the bottom of the screen.
