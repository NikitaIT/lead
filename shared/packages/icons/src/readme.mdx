## Adding new icon

### Before you start

Make sure that:

- Icons that you want to add are [not already in our pack](/story/icons--overview).
- Every icon must comply with our [icon rules][icon_rules]
  - these icon aspects can be updated through [icomoon.io][icomoon]. Please continue to steps down bellow.

### Steps

1. Get and svg icon/s from designers
2. Add your icon/s to [icomoon.io][icomoon] and make desired changes to match [visual rules][our_visual_rules]. [(icomoon docs here)][icomoon_docs]
3. Export your updated icons and save your set for easy access in the future.
4. Clone or go to repo [{pkg.name}][git_project] and add or replace new icon/s.
5. Follow instruction in that [{pkg.name}][git_project] repo for more info.
6. Then update package json in icons and internals.
7. Create pull request to [{pkg.name}][git_project] with name like: `new(Icons): Add Facebook icon`
8. After completion we will release your new icons.

### Rules for Icons

| Name     | Rule                                               |
| -------- | -------------------------------------------------- |
| Size     | 24x24 px                                           |
| margin   | at least 3px                                       |
| Centered | +                                                  |
| No color | Color will be automatically removed in UI package. |
| Content  | Icons can only have paths                          |
